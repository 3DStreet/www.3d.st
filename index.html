3DStreet

Web apps to visualize safer streets.

3DStreet is a set of open-source web-based tools to visualize safer streets, including a street component for A-Frame, a collection of street 3D assets, and reference demo apps.

  • See demo apps

Streetmix to A-Frame 3D Scene

Reference app that demonstrates how to take a URL of a streetmix street and load it in 3D with a skybox using A-Frame with orbit (scene rotation) controls for desktop or VR with compatible WebXR headsets. (VR - Oculus Quest supported, others not tested)

Docs and Learning Material

3DStreet GitHub Repo

This GitHub repo is the core of the project which contains the `street` A-Frame component, all assets including 3D models, and the reference Streetmix to A-Frame application.

Live Stream recording - Intro to A-Frame and 3DStreet for total beginners

This YouTube video is the first of a 3-part live streaming course that includes an introduction to the basics of HTML, how to make an A-Frame scene, and then how to use and configure the 3DStreet application to generate custom 3D street experiences.

Outline and Links to Sample Projects for Live Streams

This doc includes links to all 3 of the streaming courses as well as an outline for each, links to resources and sample projects shown on the stream.

Contributing to 3DStreet

I welcome contributions to 3DStreet. I prefer submitting a pull request to the 3DStreet GitHub repo which includes the street A-Frame component and 3D assets. I am also recruiting for specific volunteer and paid contracts below.

Paid Contracts (Project-based, application required)

  • Three.js / A-Frame Instancing and Performance - Project-based Contract - Accepting proposals
  • Three.js / A-Frame Advanced Street Geometry - Project-based Contract - Accepting proposals
  • Blender 3D technical artist for new and existing models - Hourly Contract - not available yet, expected July 2021
  • Submit a proposal for a paid contract

Available Roles - Volunteer (Through code for America)

  • Web Dev who wishes to learn A-Frame / three.js
  • User Experience Research and Design
  • Attend Code for SF meetup to volunteer
