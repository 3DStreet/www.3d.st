3DStreet

Web apps to visualize safer streets.

3DStreet is a set of open-source web-based tools to visualize safer streets, including a street component for A-Frame, a collection of street 3D assets, and reference demo apps.

Docs and Learning Material

3DStreet GitHub Repo

This GitHub repo is the core of the project which contains the `street` A-Frame component, all assets including 3D models, and the reference Streetmix to A-Frame application.

Live Stream recording - Intro to A-Frame and 3DStreet for total beginners

This YouTube video is the first of a 3-part live streaming course that includes an introduction to the basics of HTML, how to make an A-Frame scene, and then how to use and configure the 3DStreet application to generate custom 3D street experiences.

Outline and Links to Sample Projects for Live Streams

This doc includes links to all 3 of the streaming courses as well as an outline for each, links to resources and sample projects shown on the stream.
